Portal

A fully fledged and feature rich bot for Discord

A full Documentation with everything you may need can be found at portal-bot.xyz/docs

Documentation

  • Commands, a list of all Portal commands with descriptions, arguments and how to use them.
  • Text Interpreter is a program used by Portal to generate a channel’s name with four types of data
    1. Variables, are immutable and live data, that return information.
    2. Attributes, are mutable options that can be manipulated by whomever has clearance.
    3. Pipes, are mini functions you can pass text variables or attributes to, that manipulate their outcome.
    4. Structures, are conditional flow manipulators.
