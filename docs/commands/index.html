Commands

Commands are used to communicate with Portal, by typing {prefix}{command} (ex. ./about) you can access this command, and await a reply or action from Portal. Portal prefix: ./ (default)

about#

Description: Returns an about Portal message
Arguments : none
Accessible : everyone
Cooldown : -

announce#

Description: Announceces whatever is given by the writter to online members
Arguments : @title | @body
Accessible : everyone
Cooldown : 1 minute per user

announcement#

Description: Creates a new or sets the current channel as announcement
Arguments : @channel_name | @category_name
Accessible : admin
Cooldown : admin

authorise#

Description: Add role to admin roles
Arguments : !role
Accessible : admin
Cooldown : admin

authorised_roles#

Description: Displayes all authorised roles
Arguments : none
Accessible : everyone
Cooldown : admin

corona#

Description: Replies with the daily state of corona virus cases
Arguments : !country code (gr, de, us, etc)
Accessible : everyone
Cooldown : -

crypto#

Description: Replies with the current price of requested cryptocurrency
Arguments : !cryptocurrency | exchange
Accessible : everyone
Cooldown : -

deauthorise#

Description: Removes role from admin roles
Arguments : !role
Accessible : admin
Cooldown : admin

delete#

Description: Deletes x number of messsages on the text channel
Arguments : !number of messages to delete
Accessible : admin
Cooldown : admin

dj#

Description: Sets given member as a dj
Arguments : !member_id
Accessible : admin
Cooldown : admin

focus#

Description: Creates focus channel for you and your requested user
Arguments : !username @time (default 5minutes)
Accessible : everyone
Cooldown : -

force#

Description: Clones current channel in order to force-update name
Arguments : none
Accessible : admin
Cooldown : _2 minutes pe_r admin

help#

Description: Returns a help-list if specified returns specific description
Arguments : @command/@vrbl/@func/@pipe/@attr
Accessible : everyone
Cooldown : -

join#

Description: Joins current voice channel and announces events
Arguments : none
Accessible : everyone
Cooldown : 1 minute per user

joke#

Description: Replies with a joke
Arguments : @category
Accessible : everyone
Cooldown : -

leaderboard#

Description: Replies with the top 10 leaderboard
Arguments : @number of members
Accessible : everyone
Cooldown : -

leave#

Description: Removes Portal from voice channel
Arguments : none
Accessible : everyone
Cooldown : -

level#

Description: Returns your level
Arguments : none
Accessible : everyone
Cooldown : -

music#

Description: Creates a new or sets the current channel as music channel
Arguments : @channel_name | @category_name
Accessible : admin
Cooldown : admin

ignore#

Description: Create ignore channel or sets as ignore channel
Arguments : !channel_name | @category_name
Accessible : admin
Cooldown : admin

ignored#

Description: Displays all ignored members
Arguments : none
Accessible : everyone
Cooldown : none

ping#

Description: Returns round trip latency
Arguments : none
Accessible : everyone
Cooldown : -

portal#

Description: Create portal channel or sets as portal channel
Arguments : !channel_name | @category_name
Accessible : admin
Cooldown : admin

prefix#

Description: update the prefix of the server
Arguments : !channel_name | @category_name
Accessible : admin
Cooldown : admin

ranks#

Description: Returns ranks of ranking system
Arguments : none
Accessible : everyone
Cooldown : -

role_assigner #

Description: Replies with a message that gives roles when an emote is added
Arguments : role json
Accessible : admin
Cooldown : admin

roll#

Description: Roll follows the same philosophy as in roll20
Arguments : !roll sequence
Accessible : everyone
Cooldown : -

run#

Description: Runs the given command string and returns its output
Arguments : !exec_command
Accessible : everyone
Cooldown : -

set_ranks#

Description: Sets roles that will be given when said level reached
Arguments : rank json
Accessible : voice-portal owner, admin
Cooldown : admin

set#

Description: Sets the value of an attribute
Arguments : !attribute !value
Accessible : voice-portal owner, admin
Cooldown : -

setup#

Description: Creates an announcement, spotify, url-only and music channel
Arguments : none
Accessible : admin
Cooldown : 10 minutes per admin

spotify#

Description: Creates a new or sets the current channel as spotify channel
Arguments : none
Accessible : none
Cooldown : admin

state#

Description: Returns the current state of portal
Arguments : @channel_name | @category_name
Accessible : admin
Cooldown : -

url#

Description: Creates a new or sets the current channel as url-only
Arguments : @channel_name | @category_name
Accessible : admin
Cooldown : admin

weather#

Description: Replies with the current wheather for the requested city
Arguments : !city_name
Accessible : everyone
Cooldown : -

whoami#

Description: Replies with the your portal stats
Arguments : none
Accessible : everyone
Cooldown : -
